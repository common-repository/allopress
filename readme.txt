== Description ==

AlloPress est un plugin de monetisation, vous pouvez rentabiliser vos articles en rendant accessible une partie de leur contenu
payant (protection par systeme (c)Allopass)
Vous pouvez proteger n'importe quel type de contenu : photos, videos, texte..

Ceci est la version gratuite d'AlloPress (AlloPress Argent), pour une version plus complete rendez-vous sur le site officiel du plugin:
[AlloPress.com](http://www.allopress.com/)
La version complete gere les abonnes CB et vous permet entre autres de creer de veritables zones membres sur vos blogs.


AlloPress is a french monetization plugin, you can make any post 'pay per read' by (c)Allopass gateway (Phone and SMS payment)


== Installation ==


1. Uploadez les fichiers de l'archive dans le repertoire `/wp-content/plugins/`
2. Activez le plugin
3. Rendez-vous dans la section 'Reglages / AlloPress' de votre administration pour le configurer
    Pour la notice de configuration rendez-vous ici : http://www.allopress.com/installation
4. Pour proteger le contenu de votre article, ou une partie de ce contenu, inserez le entre les balises [ALLOPRESS] et [/ALLOPRESS]
